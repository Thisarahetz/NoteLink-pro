## Getting Started

1. **Run the Development Server:**

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

2. **Migration:**

   ```bash
   npx prisma migrate dev --name init
   ```

3. **View and Edit Your Database:**
   ```bash
   npx prisma studio
   ```
